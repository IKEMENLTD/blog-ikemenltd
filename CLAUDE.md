# Claude Code Rules for IKEMEN LTD Blog

## Emoji Usage Policy

**ABSOLUTELY NO EMOJIS**

- Do not use any emojis in any files, including HTML, Markdown, JavaScript, CSS, or any other file types
- Do not use emojis in commit messages
- Do not use emojis in code comments
- Do not use emojis in UI text, article content, or meta descriptions
- This applies to all Unicode emoji characters
- This also applies to SVG-based emoji replacements

## What to Use Instead

- Use plain text

## Why This Rule Exists

- Professional appearance
- Better accessibility
- Consistent rendering across platforms
- Easier maintenance
- Better SEO

## Enforcement

All pull requests containing emojis will be rejected. Please review your changes before committing to ensure no emojis are present.
